# random-messages

When using this package, it will get a random name from the list I made

list:
    "jordi",
    "Carolina",
    "Paula",
    "Anna",
    "Nicolau",
    "Yesica",
    "Laura"

## Install

```npm
npm install -g jordidvlop10-random-names
```

# Usage

```bash
jordidvlop10-random-names
```
